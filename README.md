# Campus Mart: The AI Bazaar

An intelligent marketplace platform designed to connect students and faculty within a university campus. This "AI Bazaar" facilitates the buying, selling, and trading of goods and services, using AI-powered features to enhance user experience and promote sustainability.

## ðŸš€ Key Features

* **User Authentication:** Secure registration and login for all campus members.
* **Product Listings:** Users can easily list items (books, electronics, furniture, notes) for sale or trade.
* **AI-Powered Recommendations:** Smart suggestions for products and services based on user's profile, major, and browsing history.
* **Smart Search:** AI-enhanced search functionality to find relevant items and services quickly.
* **Integrated Chat:** Real-time messaging for buyers and sellers to communicate, negotiate, and arrange meetups.
* **Sustainable Focus:** Features to encourage swapping, borrowing, and upcycling of campus essentials.

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Deeks779/Campus_Mart_AI_Bazar.git](https://github.com/Deeks779/Campus_Mart_AI_Bazar.git)
    cd Campus_Mart_AI_Bazar
    ```

2.  **Set up the backend:**
    *(Example for a Python/Flask backend)*
    ```bash
    cd backend
    python -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

3.  **Set up the frontend:**
    *(Example for a React frontend)*
    ```bash
    cd frontend
    npm install
    ```

4.  **Configure environment variables:**
    * Create a `.env` file in the backend root.
    * Add the following required variables:
        ```ini
        DATABASE_URL=[your_database_url]
        SECRET_KEY=[your_secret_key]
        AI_API_KEY=[your_api_key]
        ```

5.  **Run database migrations:**
    *(Example for Flask-Migrate)*
    ```bash
    flask db upgrade
    ```

### Running the Application

1.  **Start the backend server:**
    ```bash
    cd backend
    flask run
    ```

2.  **Start the frontend server:**
    *(In a new terminal)*
    ```bash
    cd frontend
    npm start
    ```
    > Open `http://localhost:3000` to view the application in your browser.
